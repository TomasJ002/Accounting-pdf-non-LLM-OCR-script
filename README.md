# Accounting-pdf-non-LLM-OCR-parser
Parsed data from national accounting repositories and realized all of them are just scans? No worries!
This is a all local OCR based on the tesseract software that will scan through all accounting statements and generate an excell file for easy parsing
Dependencies are stored in dependencies.txt
Also you need to install TesseractOCR and Poppler
*NOTE: The script is now set for the czech database from justice.cz, it can parse through any document, but you need to set the expression to search for, and the input and output folder*
